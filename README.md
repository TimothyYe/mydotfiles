## Introduction
This repository is used to manage all my dot configuration files, it includes:

>* Vim
>* Tmux <= V2.8
>* Zsh
>* Bash
>* NeoVim
>* Spacemacs
>* Alacritty
>* Fish
>* VSCode

My Doom Emacs configuration is [here](https://github.com/TimothyYe/doom-emacs), if you are interested.
